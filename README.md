# Ithania Canned Goods — Public Issue Tracker

This repository is used for **bug reports** and **feature requests** for the **Ithania Canned Goods** mod.

- The mod’s source code repository is **private**.
- You can still report issues here and follow progress via labels/status updates.

## Before you post
Please include:
- **Mod version**
- **Vintage Story version**
- Whether it happens on **Singleplayer** or **Multiplayer** (client/server)
- **Steps to reproduce**
- Relevant **log snippets** (or a link to a paste)

## How to report a bug
Open a new issue and choose **Bug report**:
- Include what you expected vs what happened
- Include reproducible steps if possible
- Attach screenshots/video if it helps

## How to request a feature
Open a new issue and choose **Feature request**:
- Describe the problem you’re trying to solve
- Describe your proposed solution
- Mention alternatives you’ve considered

## Logs
If you can, include the relevant section of your Vintage Story logs.
Please avoid posting personal information.

## Security / exploits
If you believe you’ve found a **dupe/exploit/security-sensitive** issue, do **not** post it publicly.
Instead, contact us via the mod’s distribution page.

## Quick links
- Issues: use the **New issue** button in this repo.
